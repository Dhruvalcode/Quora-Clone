# Quora Clone

A simple Quora-like question-and-answer web application built with Node.js, Express, and EJS.

## Features
- Ask and answer questions
- View questions and answers
- Edit and delete questions
- User-friendly interface

## Technologies Used
- Node.js
- Express.js
- EJS (Embedded JavaScript templates)
- CSS for styling
